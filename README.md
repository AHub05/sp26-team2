## Description 
> Spartan Marketplace is a student-only buying, selling, and service-sharing platform designed for UNCG students. Access requires university verification to improve safety and trust for campus transactions.
>
> Motivation: give students a safer alternative to public marketplaces for exchanging goods and services.
> Goals: reduce scams, increase campus engagement, and provide a central place to list items, services, and events.

2. Provider (the user with the provider role):
    1. Create/modify/remove provider profile - Providers create a seller profile with a photo, bio, and what they offer, and can update or delete it.
    2. Create services - Providers post listings with images, descriptions, pricing, and accepted payment methods.
    3. View customer statistics - Providers view metrics like listing views, messages received, and completed transactions.
    4. Reply to reviews - Providers reply to reviews to resolve concerns or thank customers.
